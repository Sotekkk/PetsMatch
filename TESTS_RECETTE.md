## Comment utiliser ce document

Pour chaque test :
1. Lis les étapes et fais-les dans l'ordre.
2. Observe ce qui se passe sur l'écran.
3. Coche **OK** si le résultat correspond à ce qui est décrit, ou **KO** si quelque chose cloche.
4. Si tu coches KO, note ce que tu as vu dans la colonne « Remarque ».

Tu peux tester sur l'application mobile **(APP)** ou sur le site web **(WEB)**, ou les deux si indiqué.
